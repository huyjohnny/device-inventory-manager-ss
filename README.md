# Device Inventory Manager

A full-stack inventory management system for tracking devices across multiple warehouses.  
Built with **Java Spring Boot**, **PostgreSQL**, and **React**.

---

## Features
- Full CRUD operations for devices and warehouses
- Warehouse capacity validation
- Device storage location tracking
- Responsive, intuitive UI

---

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** PostgreSQL
- **Frontend:** React (HTML, CSS, JS)
- **Version Control:** Git & GitHub
